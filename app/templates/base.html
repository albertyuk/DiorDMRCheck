{{ t("Running a reconciliation") }}

  1. {{ t("Upload the PLOG and DMR files on the home page. The Perimeter list is optional — with it, 无博主 rows are split by list membership.   2. {{ t("Check the parse preview (sheets, row counts, date window) and confirm — nothing external is called until you do.") }}
  3. {{ t("The run resolves every post link, then matches by note ID and author ID; only leftover ambiguous rows go to Claude for annotation, plus one final call that drafts the run summary — Claude never decides a verdict.") }}
  4. {{ t("Review the results — each verdict shows its evidence, any row can be overridden by hand — then download the annotated .xlsx or the JSON audit log.") }}
